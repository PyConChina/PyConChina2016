django REST 快速开发
10:10-10:55 张海波
作者简介: 阿里巴巴技术专家. 多年 python 主要语言开发经验,曾经就职于爱奇艺, 乐视网, 芒果 TV, 直接主要用 python 解决海量数据分析和计算问题. 目前就职于阿里中间件，致力于解决快速开发。
基于django restful 框架做快速 API开发, 大环境,微服务盛行的情况下 API 重要, 快速构建 API 的方法能极大提高企业生产力,快速应对需求和迭代变化, 主要讲解下 如何 设计 restful 以及如何快速开发一个 restful 的 api... 以及一些和 API 相关的衍生
Python在主机白盒安全检测中的应用
10:55-11:40 李毅为
作者简介: 阿里巴巴集团安全部资深开发工程师。曾就职于绿盟科技、百度云安全部。C/Python 程序员。负责阿里巴巴集团主机白盒安全产品、网络防护类产品研发。
安全领域有非常多优秀的 Python 工具和框架。本议题介绍主机白盒安全检测的概念和阿里巴巴在海量服务器的场景下对主机白盒安全检测的需求。深度介绍如何使用 Python 、一系列 Python 开源框架和 MongoDB 完成阿里巴巴主机白盒安全检测系统，为安全领域上层丰富的业务场景提供技术支持。
Graphene让Web应用迭代加速
11:40-11:50 麦伟祺
作者简介: 杭州云柚科技有限公司，曾是重庆方物科技有限公司联合创始人，作为技术负责人。
随着需求变化愈发的激烈，对迭代速度的要求也越来越高。Graphene是GraphQL的Python实现，能帮助web开发者开发出能持续快速迭代的应用。演讲会介绍GraphQL的基本原理，包括理念和基本定义；GraphQL与REST的对比；Graphene的基本用法，并用demo展示如何让Web应用的迭代速度加快。
